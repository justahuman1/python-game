# Python Game 
Lightweight snake game for your terminal. Collect the Pi's! <br/>

How to Run <br/>
==================

1.) Install Python (3) <br/>
2.) Open Terminal <br/>
3.) Type "cd" and drag the folder ("python-game-master") containing this file into the terminal <br/>
4.) Type 'python snake.py' <br/>
